PROFILE

Halo semuanya saya A. HAIDAR HAFIZ berumur baru 19tahun
saya memiliki beberapa hobi yaitu sebagai berikut :

Ini adalah salah satu hobi saya yaitu,
bermain futsal karena futsal adalah olahraga
yang menyenangkan dan menyehatkan badan
kita, bermain futsal juga mempererat
pertemanan karena di dalam futsal
membutuhkan strategi dan kekompakkan
tim, sehingga dapat memenangkan
pertandingan, jika tim tidak kompak maka
peluang kalah lebih besar.

Selanjutnya adalah game, game yang saya
sukai ialah pung, ml, dan genshin. Dalama
bermain game ini rasanya menyenangkan
karena kita bisa berseru-seruan bersama
teman ataupun sendirian. Saya bermain
game hanya ketika bosan ataupun di ajak
teman, saya tidak terlalu berlebihan dalam
bermain game karena bermain game ini
hanya untuk menghilangkan rasa bosan
ataupun jenuh.

Saya semenjak mulai kuliah menyukai
percodingan, saya mulai mempelajarinya
seperti membuat web dan membuat
menu login. saya belum mempelajari lebih
lanjut sehingga hanya beberapa kode yang
saya ingat saja, tetapi pada tahap slanjutnya
saya akan mempelajari lebih banyak lagi
supaya lancar dalam memasukkan
kode-kode tersebut.
